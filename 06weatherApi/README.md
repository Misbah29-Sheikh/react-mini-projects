# Weather App

A simple Weather App built with React and Tailwind CSS that allows users to search for a city and view current weather information.

## Features

* Search weather by city name
* Display current temperature in Celsius
* Display weather condition
* Display humidity
* Display wind speed (km/h)
* Weather condition icon
* Loading state while fetching data
* Responsive UI with glassmorphism design

## Technologies Used

* React
* Tailwind CSS
* WeatherAPI
* JavaScript
* Vite
