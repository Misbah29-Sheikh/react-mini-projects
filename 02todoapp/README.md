# Todo App (React) 📝

A simple and interactive todo application built using React.

---

## ✨ Features
- Add new tasks
- Delete tasks
- Edit tasks
- Mark tasks as completed
- Filter tasks (All / Completed / Pending)

---

## 🧠 Concepts Used
- useState
- Conditional rendering
- Array methods (map, filter)
- Controlled inputs
- Component state management
