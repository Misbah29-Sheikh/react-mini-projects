# Notes App

A modern Notes App built with React and Context API. Users can create, view, and delete notes with persistent storage using Local Storage.

## Features

* Add notes with a title and description
* Delete notes instantly
* Responsive masonry-style notes layout
* Context API for state management
* Data persistence using Local Storage
* Modern UI built with Tailwind CSS
* Responsive design for mobile and desktop

## Technologies Used

* React
* Context API
* Tailwind CSS
* Local Storage
