# Color Picker UI 🎨

A modern React UI project that changes background color dynamically based on user selection.

---

## ✨ Features
- Click to change background color
- Responsive grid layout
- Glassmorphism UI design
- Active selection highlight
- Dynamic text color adjustment (dark/light)

---

## 🧠 Concepts Used
- useState
- Dynamic inline styling
- Conditional rendering
- Tailwind CSS
- UI state management

---

## 🎨 UI Highlights
- Modern glass effect card
- Smooth hover animations
- Clean grid-based layout

---

## 🚀 Future Improvements
- Custom HEX color input
- Save favorite colors
- Copy color code button
- Gradient background mode
