# AuthApp 🔐

A simple authentication application built using React, React Router, Context API, and Local Storage.

## Features

* User Signup
* User Login
* Logout functionality
* Protected Dashboard route
* Authentication using Context API
* Persistent login using Local Storage
* Conditional Navbar links
* Responsive UI with Tailwind CSS

## Technologies Used

* React
* React Router DOM
* Context API
* Local Storage
* Tailwind CSS

## Authentication Flow

1. Users can create an account using the Signup page.
2. User information is stored in Local Storage.
3. Login validates the username and password.
4. Successful login updates the authentication state.
5. Dashboard access is restricted to authenticated users.
6. Logout clears authentication data and redirects to the Home page.
